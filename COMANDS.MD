#Notes
git init                - Tworzy nowe repozytorium w mowym projekcie
git clone               - Pobieramy zdalne repozytorium
git status              - Sprawdzamy stan repozytorium
git add                 - Dodajemy pliki do śledzenia
git rm --cached ...     - Wycofanie pliku ze śledzenia
git commit              - Wysłanie zmian do lokalnego repozytorium
git restore ...         - Przywraca konkretny plik do wersji z repozytorium
git log                 - Wyświetlenie wszystkich comitów, które wykonaliśmy w repozytorium
git revert              - Wycofanie określonego comita (nowy comit)
git show ...            - Pokazanie zmian (trzeba znać chasz cimmita)
git push                - publikuje lokalne zmiany(wysyłam je do repozytorium zdalnego)
git pull                - Pobieram zmiany ze zdalnego repozytorium
